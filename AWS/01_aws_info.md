### 지역(Region)과 가용영역(AZ: Availability Zone)
- Region : 지역 
  + aws 서버의 물리적인 위치.
  + 개발하는 위치와 주요 소비자 위치 고려해서 선정.
  + http://www.cloudping.info
- AZ(Availability Zone) : 가용영역
  + 리전 안에 있는 IDC(데이터센터)
  + 기본적으로 각 리전안에는 2개 이상의 가용 영역이 있음.
  + IDC는 각각 멀리 떨어져 있어 각종 재해시 다른 가용영역에서 서비스를 계속 할 수 있음.
