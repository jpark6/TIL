  ```python
  def quick_sort(arr):
    if len(arr) <= 1: return arr
    pivot = arr[0]
    left = [item for item in arr if pivot > item]
    mid = [item for item in arr if pivot == item]
    right = [item for item in arr if pivot < item]
    return quick_sort(left) + mid + quick_sort2(right)
  ```
